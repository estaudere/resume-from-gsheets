Create a resume from a Google sheet

First, create a properly styled Google sheet following this template , then copy the URL of the sheet into the text box below (make sure your sheet is set to "Anyone with the link can view"). Click the button to preview it as a PDF. If you'd like to preview it as a website, simply append the sheet ID to the end of the URL above. (You can test the app by clicking the "Get as PDF" button to use the sample sheet.)
